# Workplannerapp

This is the 7th class challenge in order to create a calendar application that allows a user to save events for each hour of the day. In order to exect this, the application will be done with HTML and CSS powered by jQuery.

## Steps taken:

- Created individual folders for HTML, Javascript and CSS to organise code.
- Used moment to add current day in console

## Starter code included:
- HTML 
- CSS with styling
